## 📦 Instalación

### Backend
\`\`\`bash
cd backend
python -m venv venv
venv\Scripts\activate  # Windows

\`\`\`

### Frontend
\`\`\`bash
cd frontend
npm install
\`\`\`

### Desarrollo
\`\`\`bash

cd frontend
npm start
\`\`\`

### Producción
